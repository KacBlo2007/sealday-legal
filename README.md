## About Seal Day

Seal Day is a minimalist iOS app designed as a short daily closing ritual.

- One daily reflective prompt  
- Optional single-line note  
- Press & hold interaction to “seal” the day  
- On-device storage only  
- No ads, no tracking, no accounts  

Subscriptions and payments are handled entirely by Apple via the App Store.
